Introduction

    Ce programme est utilis pour crer des vnements pour votre calendrier Outlook, vous permettant de savoir quand envoyer toutes les pr-communications menant vos projets, ainsi que tous les hypercare.

Mode d'Emploi

  1. Slectionnez le type de projet, entrez le numro du magasin, choisissez la semaine et slectionnez une langue

  2. Cliquez sur le bouton "Crer" pour gnrer une table de projet, ou cliquez sur "Annuler" pour rinitialiser le formulaire

  3. Une fois la table de projet cre, vous pouvez cliquer sur le nouveau bouton "Gnrer des vnements de calendrier", vos vnements seront enregistrs dans votre dossier de tlchargements sous la forme d'un seul fichier iCalendar (Projet - Magasin XXXX.ics)

  4. Ouvrez le fichier iCalendar et Outlook vous demandera de les ajouter votre calendrier, voil

  5. Cliquez sur le bouton "Programmer un autre projet" pour redmarrer
