Introduction

    Ce programme est utilisé pour créer des événements pour votre calendrier Outlook, vous permettant de savoir quand envoyer toutes les pré-communications menant à vos projets, ainsi que tous les hypercare.

Mode d'Emploi

  1. Sélectionnez le type de projet, entrez le numéro du magasin, choisissez la semaine et sélectionnez une langue

  2. Cliquez sur le bouton "Créer" pour générer une table de projet, ou cliquez sur "Annuler" pour réinitialiser le formulaire

  3. Une fois la table de projet créée, vous pouvez cliquer sur le nouveau bouton "Générer des événements de calendrier", vos événements seront enregistrés dans votre dossier de téléchargements sous la forme d'un seul fichier iCalendar (Projet - Magasin XXXX.ics)

  4. Ouvrez le fichier iCalendar et Outlook vous demandera de les ajouter à votre calendrier, voilà

  5. Cliquez sur le bouton "Programmer un autre projet" pour redémarrer
